# 微小宝React项目模板

# How to use

```bash
# 安装依赖
npm install
# 开发
npm start
# 构建
npm run build
```

# Features

- [x] Hot Module Replacement
- [x] Babel for React & ES6
- [x] ESLint checkout
- [x] 样式处理
- [ ] 图片处理
- [x] JS 压缩
- [x] CSS 分离
- [ ] 模块化 CSS
- [ ] SourceMap 代码映射
- [x] 开发环境 Dashboard 界面
- [x] antd按需加载(babel-plugin-import)
