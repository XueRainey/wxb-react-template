# 微小宝React项目模板

### 使用方法

```bash
# 安装依赖
yarn install
# 开发
yarn start
# 构建
yarn build
```

如果你真的不想尝试一下yarn的话，你也可以继续使用npm

```bash
# 安装依赖
npm install
# 开发
npm start
# 构建
npm run build
```

### 功能列表

- [x] Hot Module Replacement
- [x] Babel for React & ES6

- [x] webpack 资源包支持
- [x] 样式处理
- [ ] 图片处理
- [x] JS 压缩
- [x] CSS 分离
- [ ] 模块化 CSS
- [ ] SourceMap 代码映射
- [x] 开发环境 Dashboard 界面
- [x] antd按需加载(babel-plugin-import)
